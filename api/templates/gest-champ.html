Gestionnaire de championnat

Contexte

Mon premier projet pour l'épreuve E5, est une application web qui permet, comme son nom l'indique, de créer et gérer un tournoi/championnat de volley. Je l'ai créé dans le but d'aider mon club de volley lors des évenements comme le tournoi de Noël, ou ceux de beach volley l'été.
